# 📅 June 2026 Calendar

A simple, static HTML calendar for **June 2026**, styled with inline CSS.

## ✨ Features

- 🗓️ Clean table-based calendar layout
- 🔴 Sundays and Saturdays highlighted in red
- ⬛ Current date (13th) highlighted with a dark background
- 🎨 Bold, large-format date numbers for easy readability
- 📱 Responsive padding and rounded table corners with shadow

## 🚀 Usage

1. Clone or download this repository
2. Open `calender.html` in any web browser
3. That's it — no dependencies required! 🎉

## 🛠️ Tech Stack

- 🧱 HTML5
- 🎨 CSS (inline & internal styling)
